# Take-Home Assignment: Create a Styled Travel Blog Page

## Assignment Title:

**“My Dream Destination – Travel Blog Webpage”**

---

# Instructions for Students

Create a travel blog webpage about a country or city you would like to visit.

You must demonstrate strong HTML structure and correct CSS usage.

You must use:

* External CSS
* Internal CSS
* Inline CSS (only once)
* At least 5 different HTML elements beyond paragraphs
* At least 4 different CSS selectors
* CSS comments
* Proper cascading order

---

# Project Structure

```
travel-blog/
│
├── index.html
└── styles.css
```

---

# Part 1: HTML Requirements (index.html)

Your page must include:

## 1. Proper Document Structure

* `<!DOCTYPE html>`
* `<html>`
* `<head>`
* `<body>`
* Page `<title>`
* Link to external stylesheet

---

## 2. Semantic Layout Elements

You must use ALL of the following:

* `<header>`
* `<nav>`
* `<main>`
* `<section>`
* `<article>`
* `<footer>`

---

## 3. Content Requirements

Your page must include:

1. A main heading (`h1`) – Name of the destination
2. At least two `h2` subheadings
3. At least four paragraphs
4. One unordered list (things to do)
5. One ordered list (travel tips or itinerary steps)
6. One image (`img`) with:

   * `src`
   * `alt`
7. One external link (`a`)
8. One `div` container
9. One `span` element
10. One table with:

    * At least 2 rows
    * At least 2 columns
    * Table headings (`th`)

---

## 4. Attribute Requirements

Your HTML must include:

* One element with an **id**
* One element with a **class**
* One element with **two classes**
* At least one `target="_blank"` link

---

# Part 2: CSS Requirements (styles.css)

Your stylesheet must include:

### 1. Universal Selector

Add margin reset or padding to all elements.

### 2. Element Selector

Style all paragraphs or all list items.

### 3. ID Selector

Style one specific section differently.

### 4. Class Selector

Style multiple elements with a shared class.

### 5. Grouping Selector

Style `h1, h2` together.

### 6. Descendant or Combination Selector

Example:
`nav a`
or
`section p`

---

# Part 3: CSS Application Rules

You must demonstrate:

## External CSS

Most styles must be inside `styles.css`.

## Internal CSS

Inside a `<style>` tag in the head:

* Change the color of one heading.

## Inline CSS

Use inline styling on ONE element only.

---

# Part 4: Cascading & Specificity Demonstration

You must show:

* A paragraph styled using a class
* The same paragraph overridden using an id OR inline style

This demonstrates:

Inline > Internal > External
ID > Class > Element

---

# Part 5: Comments

You must include:

* At least 4 CSS comments
* At least 2 HTML comments

---

# Part 6: Design Requirements

Your page must include:

* Background color for the page
* Styled navigation bar
* Styled table (border + padding)
* A content box with padding and border
* At least one hover effect

---

# Common Errors to Avoid

* Missing semicolons
* Missing curly braces
* Incorrect property names
* Writing values like `20 px`
* Forgetting to close HTML tags
* Not adding alt text to images
