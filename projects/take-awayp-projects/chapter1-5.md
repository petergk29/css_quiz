#  Take-Home Assignment: Build Your Own Styled Webpage

## Assignment Title:

**“About Me – Fully Styled Website”**

---

##  Instructions for Students

Create a small personal webpage about yourself and style it using everything we learned about CSS.

You must use:

*  External CSS
*  Internal CSS
*  Inline CSS
*  At least 3 different selectors
*  CSS comments
*  Proper syntax (no errors!)

---

##  Project Structure

```
about-me/
│
├── index.html
└── styles.css
```

---

#  Part 1: HTML Requirements (index.html)

Your page must include:

1. A main heading (`h1`) with your name

2. At least **3 paragraphs**

3. One unordered list (`ul`) with 3 hobbies

4. One `div` section

5. At least one element with:

   * an **id**
   * a **class**
   * two classes on the same element

6. A link to your external stylesheet inside the `<head>`

---

#  Part 2: CSS Requirements (styles.css)

Your CSS file must include:

### 1️⃣ Element Selector

Style all paragraphs.

### 2️⃣ ID Selector

Style one specific element differently.

### 3️⃣ Class Selector

Style multiple elements using a class.

### 4️⃣ Universal Selector

Add margin or padding to all elements.

### 5️⃣ Grouping Selector

Style `h1, h2` together.

---

#  Part 3: CSS Application Rules

You must demonstrate:

###  External CSS

Most styling should be in `styles.css`.

###  Internal CSS

Inside `<style>` in the head:

* Change the color of one heading.

###  Inline CSS

Use inline styling on ONE element only.

---

#  Part 4: Comments

Add:

* At least **3 CSS comments**
* At least **1 HTML comment**

Example:

```css
/* This styles the main heading */
```

---

#  Part 5: Avoid These Errors

Make sure you:

* End each declaration with a semicolon ;
* Close all curly braces { }
* Do not invent property names
* Do not put spaces in values like `20 px`

---

#  Bonus (Optional – Extra Credit)

* Add a hover effect
* Use a background color for the whole page
* Create a styled box with padding and border
* Show cascading order by overriding a style
